## Base de Datos

### Crear Base de Datos y Tabla `empresa`

```sql
CREATE DATABASE prog;

USE prog;

CREATE TABLE empresa (
    id BIGINT AUTO_INCREMENT PRIMARY KEY,
    nombre VARCHAR(255),
    razon_social VARCHAR(255),
    cuit INT
);
```
